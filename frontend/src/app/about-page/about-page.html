E-Commerce Platform

A full-stack, production-ready microservices-based e-commerce platform built with Spring Boot, Angular, and modern DevOps practices.

Technologies

  • Frontend: Angular
  • Backend: Java (Spring Boot microservices)
  • Database: MongoDB
  • Media storage: Firebase Storage (used by media-service)
  • DevOps: Jenkins, SonarQube, ngrok, Nexus

Architecture

The application follows a microservices architecture. Backend services include:

  • api-gateway
  • user-service
  • product-service
  • media-service (uploads images to Firebase)
  • order-service

Run the Project

Prerequisites: Java 21+, Maven, Node.js & npm, Docker & Docker Compose, MongoDB.

Start backend & infra from project root:

make up
# or
docker compose up -d

Start frontend:

cd frontend
npm install
npm start

CI / CD

The repository includes a `Jenkinsfile` that builds services, runs SonarQube analysis and publishes Maven and Docker artifacts to Nexus.
